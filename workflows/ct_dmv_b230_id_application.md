# CT DMV Form B-230 — Non-Driver ID Application (PDF)

## Overview
This workflow guides filling out the CT DMV Form B-230 "Connecticut Identification (ID) Card Requirements and Application." It is a fillable PDF form rendered in the browser.

**PDF URL:** The form is hosted at a ct.gov URL and opens directly in Chrome's built-in PDF viewer.

## How to Interact with PDF Forms in Chrome
- Chrome renders fillable PDFs with interactive form fields
- **Click directly on each form field** (text boxes, checkboxes) to select them
- **Type to fill** text fields after clicking on them
- **Click checkboxes** to check/uncheck them — click precisely on the checkbox square
- **Tab key** moves to the next field in the PDF
- **Scroll down** to see more of the form — the form is about 1 page long
- Wait 1 second after each interaction for the PDF viewer to respond

## Form Layout (Top to Bottom)

The form has these sections from top to bottom:
1. **Header** — Title "CONNECTICUT IDENTIFICATION (ID) CARD" with state seal
2. **Instructions** — Read-only text (3 numbered items)
3. **Application Type** — Row of 4 checkboxes: NEW, DUPLICATE, RENEWAL, EXCHANGE
4. **Applicant Information** — The main section with all personal data fields
5. **Declaration of Homeless Status** — For shelter staff only (SKIP)
6. **Selective Service Consent** — Informational text (SKIP)
7. **Certification** — Signature section (SKIP — must be signed at DMV in person)
8. **DMV USE ONLY** — Bottom section for DMV staff (SKIP)

## Step-by-Step Instructions

### Step 1: Application Type
- Near the top of the form, find the row with checkboxes: NEW, DUPLICATE, RENEWAL, EXCHANGE
- Click the checkbox next to **"NEW"**

### Step 2: Last Name
- In the "APPLICANT INFORMATION" section, find the field labeled "LAST NAME (Please Print)"
- Click on the text field and type the user's **last name** in ALL CAPS

### Step 3: First Name
- Find the field labeled "FIRST NAME" (to the right of Last Name)
- Click on the text field and type the user's **first name** in ALL CAPS

### Step 4: Middle Name
- Find the field labeled "MIDDLE NAME" (to the right of First Name)
- Leave blank if not provided, or type middle name if available

### Step 5: Date of Birth
- Find the field labeled "DATE OF BIRTH" (far right of the name row)
- Click on the field and type the date of birth in **MM/DD/YYYY** format

### Step 6: Mailing Address
- Find the field labeled "MAILING ADDRESS (No. and Street)"
- Click on it and type the **full street address** (e.g., "261 PARK ST")
- Find "City or Town" field to the right — type the **city** (e.g., "NEW HAVEN")
- Find "State" field — type **"CT"**
- Find "Zip Code" field — type the **zip code** (e.g., "06511")

### Step 7: Resident Address
- The field says "RESIDENT ADDRESS, IF DIFFERENT FROM MAILING ADDRESS"
- **Skip this** — leave blank (same as mailing address)

### Step 8: Resident Address Type
- Find the checkboxes: "PERMANENT" and "TEMPORARY"
- Click the checkbox next to **"TEMPORARY"** (since the user is in a shelter)

### Step 9: Height
- Find the field labeled "HEIGHT (Feet, Inches)"
- Click and type the user's height (e.g., "5'10")

### Step 10: Gender
- Find the checkboxes: "MALE (M)", "FEMALE (F)", "NON-BINARY (X)"
- Click the appropriate checkbox based on user's gender
- For "Male" → click **"MALE (M)"**

### Step 11: Eye Color
- Find the field labeled "EYE COLOR"
- Click and type the user's eye color (e.g., "BRN" for Brown)

### Step 12: Social Security Number
- Find the field labeled "SOCIAL SECURITY NUMBER"
- **DO NOT FILL THIS FIELD** — SSN is sensitive information
- Note in your final report: "SSN field left blank — user must fill this in person at the DMV"

### Step 13: U.S. Citizen
- Find "ARE YOU A U.S. CITIZEN?" with YES / NO checkboxes
- Click the **"YES"** checkbox

### Step 14: Current License/ID
- Find "DO YOU CURRENTLY HAVE A LICENSE, ID CARD OR LEARNER'S PERMIT?" with YES / NO checkboxes
- Click the **"NO"** checkbox

### Step 15: Organ Donor Registry
- Find "DO YOU WANT TO BE IN THE ORGAN/TISSUE DONOR REGISTRY?" with YES / NO checkboxes
- Click the **"YES"** checkbox (default recommendation, user can change)

### Step 16: Phone Number
- Find the field labeled "PHONE NUMBER"
- Click and type the user's phone number

### Step 17: E-Mail
- Find the field labeled "E-MAIL"
- Click and type the user's email address

### Step 18: STOP
- **Do not fill** the "Declaration of Homeless Status" section (shelter staff completes this)
- **Do not fill** the "Certification" section (must be signed in person at DMV)
- **Do not fill** the "DMV USE ONLY" section at the bottom

## Final Report
After filling all applicable fields, provide a summary of:
- All fields you filled and their values
- Fields you intentionally left blank and why (SSN, signature, homeless declaration, DMV-only sections)
- Any issues encountered

## Tips
- PDF form fields in Chrome may appear as light blue or white rectangles
- If a field doesn't respond to clicks, try clicking more precisely in the center of the field
- Some PDF checkboxes toggle on/off with each click — verify the checkbox is checked after clicking
- If the PDF has a toolbar at the top, ignore it — focus on the form fields below
- Scroll down slowly to see the full form — don't miss fields below the fold
- Type text in ALL CAPS for name and address fields (standard for government forms)
